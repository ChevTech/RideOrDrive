RideOrDrive Cover Page

Description

A travel hub for users to post carpool ride requests and submit carpool drive offers. After signing up, users can review these requests and offers that reside in a "travel wall" and interact with each other to collaborate and plan trips together.

Target Audience

The target audiences for this application are drivers who are willing to offer a ride to others and riders who would like to request for a ride.
