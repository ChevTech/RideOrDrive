Search for a Drive Use Case

Cover Page

Title: Allow users to search for a driver.

Actor: All users have the option to search for a drive.

Prerequisite: The user needs to have an account and be signed-in: Register, Login.

Action:

  1. Complete the search form
  2. Click the search button

Effect: The user recieves all driver posts available that match his search critiria.
