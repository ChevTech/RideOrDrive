View rider comments

Title: View comments about the rider.

Actor: Any user will be able to view the comments.

Prerequisites: The user must have posted for a ride request in order to allow others to view comments about him.

Action: Click a View Comments button.

Effect: This will show all the comments about the rider from drivers who have offered him a ride.
