Search for a Ride Use Case

Cover Page

Title: Allow the user to search for Riders.

Actor: All users have the option to search for a rider.

Prerequisite: The user needs to have an account and be signed-in: Register, Login.

Action:

  1. Complete the search form
  2. Click the search button

Effect: The user recieves all rider posts available that match his search critiria.
