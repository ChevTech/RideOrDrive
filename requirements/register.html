Register Use Case

Cover Page

Title: Provide the client with the ability to sign up.

Actor: Users visiting the website and are interested in creating an account.

Prerequisite: None

Action:

  1. The user clicks on the sign-up button located on the homepage.
  2. The user enters the information required for an account.
  3. The user clicks the sign-up button.

Effect: The user creates an account and is able to login

Next: The user gets re-directed to login page.
