Post a Drive Use Case

Cover Page

Title: Allow users to post that they are offering a ride.

Actor: All users have the option to offer a ride.

Prerequisite: The user needs to have an account and be signed-in: Register, Login.

Action:

  1. Click the offer a ride button.
  2. Fill out the required information for an offer.
  3. Submit the offer

Effect: The user posts an offer for a ride.

Next: The user gets redirected to the home page.
