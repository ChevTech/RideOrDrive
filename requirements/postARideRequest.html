Post a Ride Use Case

Cover Page

Title: Allow users to post that they are looking for a ride.

Actor: All users have the option to post a ride request.

Prerequisite: The user needs to have an account and be signed-in: Register, Login.

Action:

  1. Click the post a ride request button.
  2. Fill out the required information for a request.
  3. Submit the request

Effect: The user posts a ride requests to the site.

Next: The user gets redirected to the home page.
