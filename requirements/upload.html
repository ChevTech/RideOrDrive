Upload Picture Use Case

Cover Page

Title: Allow the user to upload a picture for his profile page.

Actor: All users with a profile have the option to upload a picture.

Prerequisite: The client needs to have an account, be signed in and be in their view profile page: Register, Login, View Profile

Action:

  1. Click upload picture button.
  2. Select the picture.
  3. Click upload

Effect: A picture gets uploaded to the server and included in the client's profile.
